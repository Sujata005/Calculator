## Calulator

# React Calculator

A functional and responsive calculator web application built using the React.js library. This project demonstrates a clean implementation of component-based architecture and state management in a front-end environment.

## 🚀 Live Demo
Check out the live application here: [https://sujata005.github.io/Calculator/](https://sujata005.github.io/Calculator/)

![image](https://github.com/Sujata005/Calulator/assets/112793420/20913bba-1edc-4deb-9428-597889d6b726)

## ✨ Features
- **Standard Arithmetic:** Supports addition, subtraction, multiplication, and division.
- **Modern UI:** A clean, centred interface with high-contrast display.
- **Fully Responsive:** Optimised for both desktop and mobile screen sizes.
- **Clear & Delete:** Includes functionality to clear the entire entry or delete the last character.
- **Component-Based:** Built with modular components (Display, Buttons, and Calculator).

## 🛠️ Tech Stack
- **Library:** React.js
- **Styling:** CSS3 (Custom Styles)
- **Deployment:** GitHub Pages

## 📂 Project Structure
``` text
src/
├── components/
│   ├── Buttons.js      # Layout and logic for the button grid
│   ├── Calculator.js   # Main logic and state management
│   └── Display.js      # Rendering the input/output screen
├── styles/
│   ├── Buttons.css     # Styling for individual buttons
│   ├── Calculator.css  # Layout styling for the calculator body
│   └── Display.css     # Styling for the digital display
└── App.js              # Root component 
```


### ⚙️Installation & Setup
1. Clone the repository:
```
git clone [https://github.com/Sujata005/Calculator.git](https://github.com/Sujata005/Calculator.git)
```
2. Navigate to the project directory:
``` cd Calculator ```
3. Install dependencies:
 ``` npm install ```
4. Start the development server:
``` npm start ```
